# Bizgov — Unsubscribe Page Demo

Client preview of the Business Area unsubscribe / mailing preferences page.

**Live demo:** https://zoepolansky.github.io/bizgov-unsubscribe-demo/

---

## About

Hebrew RTL landing page for managing mailing preferences across Bizgov and the Business Area portal. Built for SFMC CloudPages.

This repository contains a **static demo version** with dummy data for client review. The production file includes AMPscript that personalises the page with the subscriber's actual linked businesses.

## Features

- Two-state flow: preference form → success confirmation
- RTL Hebrew layout (desktop two-column + responsive mobile)
- Dummy businesses shown in place of live CRM data

## Notes

- Images (logo, emblem, illustration) are served from local files in this demo
- Production deployment requires SFMC Content Builder image CDN URLs
- Do not use this file in SFMC — use the production `index.html` with AMPscript
